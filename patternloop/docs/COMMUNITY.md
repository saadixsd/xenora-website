# Community playbook (phase 8)

## Sharing patterns

1. Train or edit a LoopSpec locally.
2. `patternloop export <name>` → `.agent` zip (manifest + `loopspec.yaml`).
3. Share via GitHub gist, repo subdirectory, or Discord — **review YAML before running imports from others**.

## Importing

- `patternloop import-bundle ./foo.agent`

## Challenges (optional)

- Weekly theme: "best local agent for X boring task" — participants submit `.agent` + short README, no user data.
